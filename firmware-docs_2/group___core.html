RTOS Task: LoRa Command Task.

This task processes incoming commands from the Ground Station.

  • Starts by arming the UART4 DMA (dmaBuffer).
  • Waits for a command to arrive in loraCmdQueue (put by UART4 ISR).

  • Calls processMessage to handle the command (e.g., change frequency).
  • Re-arms the UART4 DMA to listen for the next command.
  • Releases the uart4Mutex.
