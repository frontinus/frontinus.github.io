Detailed Description

General purpose portable library for driving servo motors.

Author
Francesco Abate, Giorgio Montese

Provides a servo_t structure and functions to initialize a hardware timer in PWM mode, and to command the servo to a specific angle by calculating the required pulse width (CCR value).
