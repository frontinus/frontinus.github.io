Detailed Description

Driver for the TESEO GNSS module.

Author
Francesco Abate, Giorgio Montese

This file provides an interface for communicating with the TESEO GNSS module. It includes functions for sending NMEA commands, parsing proprietary PSTM messages (like PSTMPV), and handling continuous data reception via UART DMA with Idle Line detection.
