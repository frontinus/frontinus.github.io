uint32_t servo_init ( servo_t *  servo,
uint32_t  pwm_freq,
uint32_t  target_arr 
)

Initializes the servo and its associated hardware timer.

  1. Calculates the required PSC and ARR values based on pwm_freq and target_arr.
  2. Calculates the minpos_ccr and maxpos_ccr values from the ms pulse widths.
  3. Sets the TIM_CCR pointer to the correct hardware register.
  4. De-initializes and re-initializes the HAL timer (servo->htim) in PWM mode.
  5. Configures the specified PWM channel (servo->pwm_channel).
  6. Starts the PWM signal.
    Parameters
    servoPointer to the servo_t object to initialize.
    pwm_freqThe desired PWM frequency (e.g., PWM_FREQ).
    target_arrThe target ARR value for calculation (e.g., TARGET_ARR).
    Returns
    The calculated timer counter frequency (timer_freq).

Commands the servo to move to a specific angle (in degrees).

Clamps the requested angle within the servo's min/max limits. Converts the angle to a duty cycle, then to a CCR value, and updates the timer's hardware register.

Parameters
servoPointer to the initialized servo_t object.
angleThe target angle in degrees.
Returns
The difference between the requested and the clamped angle (0 on success, non-zero if clamping occurred).
