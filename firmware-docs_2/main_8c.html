Detailed Description

: Main program body for the Marconi Flight Computer.

Author
: Francesco Abate

This file contains the main application entry point, system initialization (clocks, peripherals), RTOS object creation, and the definitions for several key RTOS tasks and system callbacks. It also serves as the central hub for integrating all other modules like GNSS, Telemetry, and Avionics Links.
