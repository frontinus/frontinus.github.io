void gnss_config ( )

Configures the TESEO module for high-speed operation and message filtering.

Attempts to set the module baud rate to 921600, re-initializes the host UART, pings the module, and sets the desired NMEA message list (to output only $PSTMPV) and enables AGPS.

Teseo-LIV3F CONFIGURATION ROUTINE Primarily used to set which messages we want to receive from the module over UART

This routine needs to be applied only once per module, but running it multiple times is not harmful.

AFTER APPLYING THIS CONFIGURATION, THE MODULE MUST BE RESET FOR IT TO APPLY.
