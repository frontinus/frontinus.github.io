Detailed Description

Defines mission-critical constants and tunable parameters for the flight software.

Author
Francesco Abate
Date
2024-12-07

This configuration header centralizes all the key numerical values that govern the behavior of the Flight State Machine (FSM), sensor processing, and control algorithms. Modifying these values will directly impact the rocket's mission profile and event triggers.
